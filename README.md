# 🚀 Symfony Docker Activity

Welcome! In this activity, you will run a **Symfony application inside a Docker container**.

Your goal is to **create your own Docker setup** — not copy one.

---

## Project Setup

Clone this repository from GitHub:

```bash
git clone https://github.com/Souri-Dev/platform-dockerfile-activity.git
cd <project-folder>
```

---

## Objective

Run this Symfony application inside a Docker container and access it in your browser.

---

## Requirements

You must:

- Create your own `Dockerfile`
- Use a PHP-based image
- Run the Symfony application inside a container
- Make the app accessible at:

```
http://localhost:9000
```

---

## Required Docker Naming Convention

- Image name: platform-your-section
- Container name : platform-your-section-container

---

## Rules

- Do NOT copy a full solution from classmates or the internet
- You may read documentation and guides
- You must understand and explain your work

---

## Expected Output

When everything is working:

- You can open a browser and go to:

```
http://localhost:9000
```

- You should see the **Congratulations Dashboard**

---

## Evaluation Criteria

- Application runs correctly in Docker
- Dockerfile is valid and clean
- You can explain your setup
- You can modify it when asked
